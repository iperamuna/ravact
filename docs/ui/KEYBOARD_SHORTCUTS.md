# Keyboard Shortcuts Reference

[← Back to Documentation](../README.md)

Complete keyboard shortcuts reference for Ravact.

## Global Shortcuts

These work on most screens throughout the application.

| Key | Action |
|-----|--------|
| `↑` / `k` | Move up |
| `↓` / `j` | Move down |
| `Enter` | Select / Confirm |
| `Esc` | Go back / Cancel |
| `q` | Quit to previous screen |
| `Ctrl+C` | Quit application |
| `c` | Copy to clipboard (where supported) |

## Tips

1. **Arrow keys always work** - Use them if you're not comfortable with vim keys
2. **Tab in forms** - Use Tab, not arrow keys, to navigate form fields
3. **c for copy** - Works on most screens with displayable content
4. **? for help** - In File Browser, shows all shortcuts
5. **Esc goes back** - Use it to navigate up the menu hierarchy
